# Daily AI Open Source

![Daily Update](https://img.shields.io/badge/update-daily-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue) ![Status](https://img.shields.io/badge/status-active-success)

## English

### Description
This repository tracks **daily trending AI open source projects** from **GitHub**, **Hugging Face**, and **Papers with Code**. It provides a concise, structured snapshot of what is popular and noteworthy each day.

### Categories
- LLM
- Computer Vision
- NLP
- MLOps
- AI Agents
- Multimodal

### Daily Update Format
```
- Date: YYYY-MM-DD
  - Project: Project Name
  - Stars: 1234
  - Description: One-line summary
  - Link: https://github.com/owner/repo
```

### Sample Entry (2026-03-23)
- Date: 2026-03-23
  - Project: OpenVision-Lite
  - Stars: 12,345
  - Description: Lightweight multimodal model for vision-language tasks.
  - Link: https://github.com/example/openvision-lite

### How to Contribute
1. Fork this repository.
2. Add a new entry in the latest daily section following the format.
3. Submit a pull request with a clear summary.

## 中文

### 说明
本仓库用于记录来自 **GitHub**、**Hugging Face** 和 **Papers with Code** 的 **每日热门 AI 开源项目**，以结构化的方式快速呈现当天值得关注的项目。

### 分类
- LLM（大语言模型）
- Computer Vision（计算机视觉）
- NLP（自然语言处理）
- MLOps
- AI Agents（智能体）
- Multimodal（多模态）

### 每日更新格式
```
- 日期: YYYY-MM-DD
  - 项目: 项目名称
  - Stars: 1234
  - 简介: 一句话描述
  - 链接: https://github.com/owner/repo
```

### 示例条目（2026-03-23）
- 日期: 2026-03-23
  - 项目: OpenVision-Lite
  - Stars: 12,345
  - 简介: 轻量级多模态模型，适用于视觉-语言任务。
  - 链接: https://github.com/example/openvision-lite

### 如何贡献
1. Fork 本仓库。
2. 按照格式在最新日期下添加条目。
3. 提交 Pull Request 并说明改动内容。
